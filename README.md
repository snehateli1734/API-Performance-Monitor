# 🚀 API Performance Monitor

## Project Description
This is a simple Python project that monitors API performance by measuring response time, status code, and request success/failure. It helps in understanding how fast and reliable an API is.

---

## Features
- Checks API response time
- Displays HTTP status code
- Shows success/failure status
- Handles errors gracefully

---

## Technologies Used
- Python
- Requests Library
